# 表值构造函数



Transact-SQL 使用`VALUES`关键字指定要构造到表中的一组行值表达式。Transact-SQL 可用于`FROM`在子句中用于在查询时构建临时表。

```sql
VALUES (row_value_list)[,...n]
```

| 参数             |                                                              |
| ---------------- | ------------------------------------------------------------ |
| `row_value_list` | 以逗号分隔的列值列表。将每个列表括在括号中，并用逗号分隔多个列表。每个列表必须具有相同数量的值，并且值必须与表中的列具有相同的顺序。每个列表必须包含每列的值。 |

使用方法如下：

```sql
SELECT
  expression[,...n]
FROM
  (VALUES (row_value_list)[,...n]) [AS] table_name(column_name[,...n])
```

使用 Transact-SQL 时，`AS`关键字是可选的，并且在命名表和提供列名时隐含。

```sql
SELECT *
FROM
  (VALUES ('2023-01-01 12:00:00'::TIMESTAMP, 1.23, 4.56),
          ('2023-01-01 13:00:00'::TIMESTAMP, 2.46, 8.1),
          ('2023-01-01 13:00:00'::TIMESTAMP, 4.81, 16.2)
  ) AS data(time, f1, f2);
```

将返回如下结果：

```sql
+---------------------+------+------+
| time                | f1   | f2   |
+---------------------+------+------+
| 2023-01-01T12:00:00 | 1.23 | 4.56 |
| 2023-01-01T13:00:00 | 2.46 | 8.1  |
| 2023-01-01T13:00:00 | 4.81 | 16.2 |
+---------------------+------+------+
```
